Document:
 The Document Object is an essential component of web browsers and represents the parsed HTML structure of a web page. 
It encapsulates the content, structure, and functionality of the loaded webpage, providing access to elements, 
styles, scripts, and other resources. 
In a typical web browsing scenario, the Document Object is responsible for rendering web content and handling user
 interactions, making it the cornerstone of the user's web experience.
